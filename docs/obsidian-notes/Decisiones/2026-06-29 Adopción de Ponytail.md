# 2026-06-29 Adopción de Ponytail

## Decisión

Adoptar [[Ponytail]] como regla global de implementación mínima para tareas de código
del [[Sistema de Trazabilidad]].

Se aplica después de leer el contexto de [[Obsidian]], consultar [[Graphify]] y
comprender el flujo real. Antes del cierre de un cambio relevante se revisa el diff
para eliminar complejidad prescindible.

## Límites

La simplificación no elimina seguridad, validación, prevención de pérdida de datos,
accesibilidad, requisitos explícitos ni una verificación mínima. Tampoco autoriza
dependencias nuevas para ahorrar líneas ni afirmaciones no medidas sobre ahorro de
tokens.

## Consecuencias

Los prompts de proyectos nuevos y existentes incluyen la regla y una alternativa
manual cuando la skill no esté disponible.

Relacionado con [[Reglas globales del Sistema de Trazabilidad]], [[Ponytail]],
[[Obsidian]] y [[Graphify]].
