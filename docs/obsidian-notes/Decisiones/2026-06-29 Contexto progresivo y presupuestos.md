# 2026-06-29 Contexto progresivo y presupuestos

## Decisión

Adoptar divulgación progresiva para todos los proyectos:

- contrato y cápsula compactos;
- [[Graphify]] con presupuesto 600 y máximo tres fuentes;
- contexto histórico bajo demanda;
- [[Ponytail]] global activado desde cada proyecto;
- sesiones de máximo 500 tokens estimados;
- [[Notion]] recibe solo trabajo y resultado, máximo 120 palabras.

Los presupuestos se verifican automáticamente. La estimación usa caracteres/4 y no se
presenta como conteo exacto del modelo.

## Motivo

El baseline estimado fue ~6300 tokens documentales al iniciar y ~2425 al cerrar. El
flujo anterior pagaba contexto irrelevante y duplicaba reglas entre prompts.

## Consecuencia

[[Obsidian]] conserva todo el conocimiento durable, pero solo se carga la evidencia
necesaria para la tarea. No se crea otro servicio ni se cambia el modelo de Notion.
